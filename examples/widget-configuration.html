ðŸ’¡ Pro Tips:
  • Use data-external-id to maintain conversation history for returning users
  • JSON arrays in data-suggestions must be properly encoded
  • Boolean attributes: use "true" or "false" strings
  • Themes: "light", "dark", or "system"
  • Colors: Use hex format like "#6366f1" for primary color
  • Positions: "bottom-right", "bottom-left", "bottom-center"
  • Sizes: "compact", "normal", "large"
  • Border radius: Any number (pixels), e.g. "16" or "24"
  • Omit attributes to use default values
